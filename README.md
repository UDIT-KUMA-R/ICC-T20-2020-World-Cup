🏏 Cricket Analysis Dashboard using Power BI
Overview

The Cricket Analysis Dashboard is an interactive Power BI project developed to explore and analyze cricket match data, player statistics, and team performance. The dashboard helps users understand trends in cricket data through visual insights and supports data-driven analysis of matches and player contributions.

🚀 Features

Player Performance Insights – Detailed analysis of batting, bowling, and fielding statistics.

Team Performance Comparison – Evaluate and compare team results across different matches and tournaments.

Match Statistics Analysis – Examine important metrics such as runs scored, wickets taken, strike rates, and economy rates.

Interactive Data Visualizations – Engaging charts and visuals that simplify complex cricket statistics.

Dynamic Filters – Users can filter data by season, team, or player to explore specific insights.

📊 Visualizations Included

The dashboard contains multiple visualization types to represent cricket data effectively:

Bar charts to compare runs, wickets, and other match statistics

Line charts to observe performance trends over time

Pie charts showing player contribution and distribution of statistics

Scatter plots for analyzing relationships such as strike rate vs economy rate

Interactive tables with sortable and filterable cricket statistics

📂 Dataset

The dashboard is built using cricket datasets that include:

Match-level statistics such as runs, wickets, and overs

Individual player performance data including batting, bowling, and fielding records

Team-level information like matches won, matches lost, and net run rate

Data Sources: Public cricket datasets such as Kaggle and ESPN Cricinfo.

🛠 Tools & Technologies

Power BI Desktop – Used to design and build the interactive dashboard

DAX (Data Analysis Expressions) – Used for creating calculated metrics and measures

Power Query – Used for data preparation and transformation

CSV / Excel Files – Used as the data sources for the dashboard

🎯 Dashboard Insights

The dashboard helps uncover several key insights such as:

Identification of top-performing batsmen and bowlers based on runs, wickets, and strike rates

Comparative analysis of team performance in head-to-head matches

Seasonal performance trends and patterns in cricket matches

Evaluation of player consistency and overall impact on matches
